Crear una ventana en Tkinter es bastante sencillo. Aquí tienes un ejemplo básico de cómo hacerlo:

python
import tkinter as tk

# Crear la ventana principal
ventana = tk.Tk()

# Configurar el título de la ventana
ventana.title("Mi Primera Ventana en Tkinter")

# Configurar el tamaño de la ventana
ventana.geometry("400x300")  # Ancho x Alto

# Agregar un widget de etiqueta
etiqueta = tk.Label(ventana, text="¡Hola, Tkinter!", font=("Arial", 14))
etiqueta.pack(pady=20)

# Iniciar el bucle principal de la ventana
ventana.mainloop()

Descripción del código:

  1. Importar Tkinter:

    python
    import tkinter as tk

    Tkinter se importa como tk para simplificar el uso de sus clases y funciones.

  2. Crear la ventana principal:

    python
    ventana = tk.Tk()

    Esto crea una instancia de la ventana principal.

  3. Configurar el título y el tamaño de la ventana:

    python
    ventana.title("Mi Primera Ventana en Tkinter")
ventana.geometry("400x300")

    Estableces el título de la ventana y defines su tamaño (400x300 píxeles).

  4. Agregar un widget:

    python
    etiqueta = tk.Label(ventana, text="¡Hola, Tkinter!", font=("Arial", 14))
etiqueta.pack(pady=20)

    Aquí se agrega una etiqueta con texto a la ventana y se posiciona automáticamente con pack().

  5. Iniciar el bucle principal:

    python
    ventana.mainloop()

    Este es el bucle principal que mantiene la ventana abierta y responde a eventos.

Para posicionar la imagen en un lugar específico de la ventana en Tkinter, puedes usar los métodos de geometría como pack(), grid(), o place(). Cada uno de estos métodos te permite controlar cómo y dónde se coloca el widget en la ventana.

Ejemplo usando pack() con side:

Si quieres posicionar la imagen en un lado específico (por ejemplo, a la izquierda, derecha, arriba, o abajo), puedes usar el método pack() con la opción side.

python
import tkinter as tk
from PIL import Image, ImageTk

# Crear la ventana principal
ventana = tk.Tk()
ventana.title("Ventana con Imagen Posicionada")
ventana.geometry("400x400")

# Cargar y redimensionar la imagen usando PIL
imagen_original = Image.open("ruta/de/tu/imagen.jpg")
imagen_redimensionada = imagen_original.resize((200, 200))
imagen_tk = ImageTk.PhotoImage(imagen_redimensionada)

# Crear un widget de etiqueta que contenga la imagen
etiqueta_imagen = tk.Label(ventana, image=imagen_tk)

# Posicionar la imagen en la parte izquierda de la ventana
etiqueta_imagen.pack(side="left", padx=20, pady=20)

# Iniciar el bucle principal de la ventana
ventana.mainloop()

Ejemplo usando place() para posicionamiento absoluto:

Si necesitas un control más preciso sobre la posición de la imagen (por ejemplo, en coordenadas específicas), puedes usar el método place().

python
import tkinter as tk
from PIL import Image, ImageTk

# Crear la ventana principal
ventana = tk.Tk()
ventana.title("Ventana con Imagen Posicionada Absolutamente")
ventana.geometry("400x400")

# Cargar y redimensionar la imagen usando PIL
imagen_original = Image.open("ruta/de/tu/imagen.jpg")
imagen_redimensionada = imagen_original.resize((200, 200))

# Crear un widget de etiqueta que contenga la imagen
etiqueta_imagen = tk.Label(ventana, image=imagen_tk)

# Posicionar la imagen en coordenadas específicas (x=50, y=100)
etiqueta_imagen.place(x=50, y=100)

# Iniciar el bucle principal de la ventana
ventana.mainloop()

Descripción de los métodos:

  • pack(side="left"): Coloca la imagen en el lado izquierdo de la ventana. También puedes usar "right", "top", o "bottom" para otros lados.

  • place(x=50, y=100): Coloca la imagen en una posición absoluta, donde x e y son las coordenadas en píxeles desde la esquina superior izquierda de la ventana.

  • padx y pady: Son márgenes internos que puedes ajustar para controlar el espacio alrededor del widget.

Ejemplo usando pack() para colocar botones al lado de una imagen:

python
import tkinter as tk

ventana = tk.Tk()

frame = tk.Frame(ventana)

etiqueta_imagen.pack(side="left", padx=10)

boton1 = tk.Button(frame, text="Botón 1")
boton1.pack(side="left", padx=10)

boton2 = tk.Button(frame, text="Botón 2")
boton2.pack(side="left", padx=10)

# Iniciar el bucle principal de la ventana
ventana.mainloop()

Descripción del código:

  1. Crear un contenedor (Frame):

    python
    frame = tk.Frame(ventana)
frame.pack(padx=20, pady=20)

    Aquí se crea un contenedor Frame para organizar la imagen y los botones. Esto facilita el control del diseño.

  2. Posicionar la imagen y los botones:

    python
    etiqueta_imagen.pack(side="left", padx=10)
boton1.pack(side="left", padx=10)

Ejemplo usando grid() para un diseño más controlado:

Si prefieres un control más preciso sobre la posición de los widgets, puedes usar grid():

python
import tkinter as tk
from PIL import Image, ImageTk

En este ejemplo, los widgets se colocan en una cuadrícula (grid) con filas (row) y columnas (column). La imagen se coloca en la primera columna, y los botones en las columnas siguientes.

import tkinter as tk
from tkinter import Toplevel, Label, Button

# Crear la ventana principal
ventana = tk.Tk()
ventana.title("Ventana principal")
ventana.geometry("600x400")

# Definir la función para el popup
def popup():
    popupwindow = Toplevel(ventana)
    popupwindow.title("Alert")
    popupwindow.geometry("200x100")
    alert = Label(popupwindow, text="¿Está correcto?")
    button1 = Button(popupwindow, text="OK", command=popupwindow.destroy)
    alert.pack(pady=10)
    button1.pack(pady=5)

# Crear el marco y un botón que abre el popup
frame = tk.Frame(ventana)
frame.pack(padx=20, pady=20)
